Permanently remove calculations and their files. Every action here shows what it will delete first and requires the exact name typed back. None of it can be undone.

Users

Each user gets one API key. They submit with it, sign in with it, and see only their own projects. A key is shown once, at creation or rotation, and is stored only as a hash — it can never be read back, only replaced.
